# 老爷爷引导系统设计

## 目标

全屏引导：可配置关卡、台词、高亮格子；每关仅首次展示；风格与现有土楼/牧场像素 UI 一致。

## 配置 `data/guide.json`

```json
{
  "guides": [
    {
      "level": 1,
      "lines": ["……", "……"],
      "highlight": { "row": 0, "col": 0 }
    }
  ]
}
```

| 字段 | 含义 |
|------|------|
| `level` | 关卡号，从 1 开始 |
| `lines` | 台词数组，点击推进一句 |
| `highlight.row/col` | 地图格子坐标，从 0 开始 |

## 交互

1. 进入关卡且该关未看过引导 → 弹出全屏遮罩
2. 遮罩抠出 `highlight` 对应格子
3. 点击推进台词；说完关闭并记入 localStorage
4. 引导中禁止滑动下棋
5. 编辑器试玩不触发；每关只播一次

## UI

参考气泡布局：绿框老爷爷头像 + 米色圆角对话框 + 柔和深绿遮罩；色调对齐游戏草绿/木色。
